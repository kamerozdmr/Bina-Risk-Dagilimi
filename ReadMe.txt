Uygulama ile taşıyıcı sistemi betonarme ve yığma olan binaların risk tespiti
yayınlanmış esaslara göre yapılmaktadır.

Uygulama Python versiyon 3.8 ile denendi.

Gerekli kütüphaneler:
streamlit	(https://docs.streamlit.io/library/get-started/installation)
Pillow (PIL)	(https://pillow.readthedocs.io/en/stable/installation.html)
Pandas

Programın Çalıştırılması:
Gerekli kütüphaneler kurulduktan sonra komut isteminden "1_Giriş.py" dosyasının
bulunduğu klasöre gidilir. "py -m streamlit run 1_Ana_Sayfa.py" komutu ile 
program çalıştırılır.
